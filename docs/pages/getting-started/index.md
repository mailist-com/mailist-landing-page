# Pierwsze Kroki z Mailist

Witaj w sekcji Pierwszych Kroków! Znajdziesz tutaj wszystkie informacje potrzebne do rozpoczęcia pracy z platformą Mailist.

## Co znajdziesz w tej sekcji?

### [Szybki Start](quick-start.html)

Przewodnik krok po kroku, który pomoże Ci:
- Skonfigurować klucz API
- Zainstalować odpowiednie biblioteki
- Wysłać pierwszy email
- Sprawdzić status wysyłki

Czas realizacji: **5-10 minut**

### [Instalacja](installation.html)

Szczegółowe instrukcje instalacji dla różnych platform:
- Node.js / JavaScript
- Python
- PHP
- Ruby
- Go
- .NET

### [Autentykacja](authentication.html)

Dowiedz się jak:
- Generować i zarządzać kluczami API
- Używać OAuth 2.0
- Zabezpieczać swoje żądania
- Obsługiwać tokeny odświeżania

### [FAQ](faq.html)

Najczęściej zadawane pytania i odpowiedzi dotyczące:
- Konfiguracji konta
- Limitów wysyłki
- Problemów z dostarczalnością
- Rozliczeniami

### [Changelog](changelog.html)

Śledź najnowsze aktualizacje, nowe funkcje i poprawki błędów w naszej platformie.
