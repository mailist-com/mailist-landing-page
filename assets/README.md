# Assets

Ten folder zawiera zasoby statyczne dla landing page Mailist.

## Struktura

```
assets/
├── images/          # Obrazy produktu, screenshots, itp.
│   ├── hero/
│   ├── features/
│   └── testimonials/
│
└── icons/          # Ikony i małe grafiki
    ├── social/
    └── features/
```

## Zalecenia

### Obrazy
- **Format**: WebP dla lepszej kompresji (fallback do PNG/JPG)
- **Rozmiar**: Maksymalnie 1920px szerokości dla hero images
- **Optymalizacja**: Użyj narzędzi jak TinyPNG, ImageOptim
- **Naming**: Używaj kebab-case (np. `hero-dashboard-preview.webp`)

### Ikony
- **Format**: SVG (skalowalny, mały rozmiar)
- **Fallback**: PNG dla starszych przeglądarek
- **Naming**: Używaj opisowych nazw (np. `icon-email.svg`)
